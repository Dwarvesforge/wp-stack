# set_login_logo_url

Set the login form logo url



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$url  |  **{ [String](http://php.net/manual/en/language.types.string.php) }**  |  The url to set  |  required  |

### Example
```php
	WPS::set_login_logo_url('https://my-cool-website.com');
```
