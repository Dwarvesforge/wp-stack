# is_visitor

Detect if the user is a visitor and not a logged in administrator or content editor


Return **{ [Boolean](http://php.net/manual/en/language.types.boolean.php) }** `true` if is a visitor, `false` if not

### Example
```php
	WPS::is_visitor();
// true
```
