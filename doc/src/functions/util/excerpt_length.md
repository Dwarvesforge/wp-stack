# excerpt_length

Change the excerpt length



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$length  |  **{ [Integer](http://php.net/manual/en/language.types.integer.php) }**  |  The new excerpt length  |  optional  |  100

### Example
```php
	WPS::excerpt_length(200);
```
